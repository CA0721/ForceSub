<details>

  <summary>VPS</summary>

<br>

- Clone this repo
```
git clone https://github.com/TheFlashSpeedster/ACE-Force-Subscribe
```
- Change directory
```
cd ACE-Force-Subscribe
```
- Install requirements
```
pip3 install -r requirements.txt
```

- Edit Config via nano
```
nano Config.py
```
- Start the bot
```
python3 bot.py
```
